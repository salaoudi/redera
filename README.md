# redera
Redera is a social network app built with firebase. 
I used firebase Auth to Authenticate users.
I used Firebase Storage to store users posts and Images.

**Tech Used**
  - SWIFT
  - SWIFTUI
  - FIREBASE
  - FIREBASE AUTH
  - FIREBASE STORAGE
